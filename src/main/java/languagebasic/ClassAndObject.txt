4. Find errors and fix if any
public class A {
    private class B {}
}

public class MainClass extends A {
    public static void main(String[] args) {
        B b = new B();
    }
}

Answer #4:
Must change the access modifier of the inner class B into protected or public to allow access from other classes.
Besides, the object creation for an inner class must also be corrected. So the fixing is:

public class A {
    protected class B {}  //or public
}

public class MainClass extends A {
    public static void main(String[] args) {
        B b = new A().new B();
    }
}

6. Find errors and fix if any
package pack1;
class A {
    public A(){}
}

package pack2;
import pack1.*;
class B {
    A a = new A();
}

Answer #6:
Can not access class A outside package pack1 because access modifier of A is in default, it means it only allows
access from classes in the same package. To fix this error, we declare a public modifier for class A as below:
package pack1;
public class A {
    public A(){}
}

package pack2;
import pack1.*;
class B {
    A a = new A();
}
